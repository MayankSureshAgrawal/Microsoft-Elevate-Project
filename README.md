🩺 MedAI Lite – AI Disease Prediction System
MedAI Lite is a simple and powerful Machine Learning web application built using Streamlit that predicts:

🩸 Diabetes

❤️ Heart Disease

🫀 Liver Disease

🚀 Features
Clean and modern UI
Fast predictions using trained ML models
Beginner-friendly interface
Works on any device

🧠 Technologies Used
 -> Python 
 -> Streamlit
 -> Scikit-learn
 -> NumPy
 
📁 Project Structure
.
├── main.py
├── requirements.txt
├── Saved_Models/
│   ├── diabetes_model.sav
│   ├── heart_disease_model.sav
│   ├── liver_disease_model.sav

⚙️ Installation
git clone https://github.com/your-username/medai-lite.git
cd medai-lite
pip install -r requirements.txt

▶️ Run the App
streamlit run main.py
