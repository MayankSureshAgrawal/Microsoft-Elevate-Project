# 🩺 MedAI Lite – AI Disease Prediction System

MedAI Lite is a simple yet powerful **Machine Learning web application** built using **Streamlit** that predicts multiple diseases based on user input.

---

## 🚀 Features

* 🩸 Diabetes Prediction
* ❤️ Heart Disease Prediction
* 🫀 Liver Disease Prediction
* ⚡ Fast and real-time predictions
* 🎨 Clean and modern user interface
* 💻 Beginner-friendly and easy to use

---

## 🧠 Technologies Used

* Python 🐍
* Streamlit 🌐
* Scikit-learn 🤖
* NumPy 🔢

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/medai-lite.git
cd medai-lite
```

2. Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```
streamlit run main.py
```
